# OpenClaw 教程网站技术实现指南

## 🛠️ 技术栈确定

### 核心技术
- **框架**: Astro (最新稳定版)
- **主题**: 自定义轻主题（基于 Astro）
- **部署**: Cloudflare Pages
- **搜索**: Pagefind (本地搜索，零配置)
- **广告**: Google AdSense
- **分析**: Google Analytics (可选)

### 为什么选择这个组合？
1. **性能**: Astro 静态生成，首屏极快
2. **SEO**: 自动 sitemap，完美的 meta 标签支持
3. **速度**: Cloudflare Pages 全球 CDN
4. **成本**: 完全免费（个人项目）
5. **开发**: 开发体验好，构建速度快

## ⚙️ 核心配置文件

### 1. `astro.config.mjs`
```javascript
import { defineConfig } from 'astro/config';
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://clawtutorial.net',
  integrations: [
    sitemap({
      changefreq: 'weekly',
      priority: 0.7,
      lastmod: new Date(),
    }),
  ],
  vite: {
    build: {
      rollupOptions: {
        output: {
          assetFileNames: 'assets/[hash][extname]',
        },
      },
    },
  },
});
```

### 2. `src/content/config.ts`
```typescript
import { defineCollection, z } from 'astro:content';

const post = defineCollection({
  schema: z.object({
    title: z.string(),
    description: z.string(),
    pubDate: z.coerce.date(),
    lastUpdated: z.coerce.date().optional(),
    slug: z.string(),
    tags: z.array(z.string()).default([]),
    difficulty: z.enum(['beginner', 'intermediate', 'advanced']).default('beginner'),
    estimatedTime: z.string().default('5 minutes'),
    prerequisites: z.array(z.string()).default([]),
    alternates: z.object({
      zhCN: z.string().optional(),
    }).optional(),
  }),
});

export const collections = {
  posts,
  'posts-zh-cn': post,
};
```

## 🚀 部署到 Cloudflare Pages

### 1. 准备工作
```bash
# 安装依赖
npm install

# 本地测试
npm run dev

# 构建测试
npm run build
```

### 2. Cloudflare Pages 设置
- **连接仓库**: GitHub → clawtutorial
- **构建命令**: `npm run build`
- **输出目录**: `dist`
- **环境变量**:
  - `NODE_VERSION`: `22`

### 3. 域名配置
- 在 Cloudflare DNS 添加 A 记录指向 Pages
- 启用自动 HTTPS
- 配置 301 重定向 (www ↔ non-www)

### 4. 构建优化
```javascript
// astro.config.mjs 添加
vite: {
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          'vendor': ['@astrojs/sitemap'],
        },
      },
    },
  },
}
```


## 🔄 持续集成

### GitHub Actions
```yaml
# .github/workflows/deploy.yml
name: Deploy to Cloudflare Pages

on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '22'
          cache: 'npm'
      - run: npm ci
      - run: npm test
      - run: npm run build
```
