## Common Issues（常见问题）

### Issue 1: "Permission denied"
**原因**：权限不足
**解决**：
```bash
sudo npm install -g openclaw
```
或使用 nvm 管理的 Node.js

### Issue 2: "Port already in use"
**原因**：端口被占用
**解决**：
```bash
# 查找占用进程
lsof -i :18789
# 杀死进程
kill -9 <PID>
```

[→ 详见：端口冲突排查](/troubleshooting/port-18789/)
